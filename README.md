# 网络通信

> *程序运行环境依赖 vue-cli*

## 程序功能
用vue通过axios实现对GitHub相关用户搜索及访问其个人主页【GitHub api：https://api.github.com/search/users?q=xxx】

## 配置运行环境

从官网上安装node.js

[Download | Node.js (nodejs.org)](https://nodejs.org/en/download/)



安装vue框架

```bash
npm i vue
```



配置环境变量

将vue.cmd添加到环境变量中



运行程序

进入程序根目录（本文件所在目录），执行

```bash
npm run serve
```
